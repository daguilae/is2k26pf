ℹ️
Carga automática

Al iniciar el formulario se ejecuta fun_CargarMovimientos() y la tabla se puebla automáticamente.

✅
Doble clic para ver detalle

Haga doble clic en cualquier fila para abrir Frm_Encabezado_Transaccion con los datos precargados. Al cerrarlo, el listado se actualiza solo.

No es posible editar celdas directamente. Toda modificación se realiza desde el formulario de detalle.

Botones y ControlesPágina 3 de 7
2 Botones y Controles

La barra de herramientas contiene los controles de acción. Algunos se habilitan o deshabilitan automáticamente según los permisos asignados al usuario activo.

➕
Ingresar
Requiere: permiso INGRESAR

Abre Frm_Encabezado_Transaccion en modo creación para registrar un nuevo movimiento desde cero.

🔄
Refrescar
Siempre disponible

Recarga el listado ejecutando fun_CargarMovimientos() nuevamente. Útil si otro usuario realizó cambios.

🖨️
Imprimir
Requiere: permiso IMPRIMIR

Abre Frm_ReporteMovimientoInventario para generar un informe impreso del listado.

📊
DashBoard / Filtrar
Requiere: permiso CONSULTAR

Abre el panel de consultas avanzadas DSH_BRD_FRM para filtrar y analizar movimientos.

⏮◀▶⏭
Navegación
Siempre disponible

Botones Inicio, Anterior, Siguiente, Fin: permiten desplazarse entre los registros del listado.

❓
Ayuda
Siempre disponible

Muestra la ayuda contextual del formulario. Funcionalidad configurable según la implementación del sistema.

🚪
Salir
Siempre disponible

Cierra el formulario ejecutando this.Close(). No solicita confirmación previa al usuario.

3 Columnas del Listado

El DataGridView principal muestra las siguientes columnas. Está configurado en modo solo lectura con ajuste automático del ancho de columnas (AutoSizeColumnsMode = Fill).

Columna (UI)Campo en BDTipoDescripción

ℹ️
Configuración del DataGridView

Propiedades activas: AllowUserToAddRows = false · ReadOnly = true · AutoSizeColumnsMode = Fill · SelectionMode = FullRowSelect.

⚠️
Datos enviados al formulario hijo

Al hacer doble clic, se extraen exactamente los campos ID, ID_Tipo, Tipo_Movimiento, Fecha y Descripcion y se pasan como parámetros a Frm_Encabezado_Transaccion.

Operaciones ComunesPágina 5 de 7
4 Operaciones Comunes

▸ Registrar un nuevo movimiento

1

Verificar que "Ingresar" esté habilitado

Si aparece deshabilitado (gris), su usuario no tiene el permiso Ingresar. Contacte al administrador del sistema.

2

Hacer clic en "Ingresar"

Se abrirá Frm_Encabezado_Transaccion en modo creación. Complete los campos: tipo de movimiento, fecha y descripción.

3

Guardar y presionar "Refrescar"

Al guardar, el formulario hijo se cierra. Presione Refrescar en el listado para ver el nuevo registro.

▸ Consultar o editar un movimiento existente

1

Localizar el registro

Desplácese con scroll, use los botones de navegación, o use Filtrar / DashBoard para búsquedas avanzadas.

2

Hacer doble clic sobre la fila

El sistema extrae los datos y los envía al formulario de detalle Frm_Encabezado_Transaccion.

3

Cerrar el formulario de detalle

Al cerrarlo, el listado principal se recarga automáticamente con los cambios.

▸ Generar reporte impreso

1

Clic en "Imprimir"

Se abre Frm_ReporteMovimientoInventario. Configure los parámetros del reporte si es necesario.

2

Vista previa y envío

Desde el visor puede previsualizar, exportar a PDF o enviar directamente a la impresora configurada.

Preguntas FrecuentesPágina 7 de 7
6 Preguntas Frecuentes

Respuestas a los casos más comunes que pueden surgir al usar este formulario.

¿Por qué el botón "Ingresar" aparece deshabilitado? ›
Su usuario no tiene el permiso Ingresar para la aplicación 728. Contacte al administrador del sistema para que le otorgue el acceso desde el módulo de seguridad.
Ingresé un movimiento pero no aparece en el listado ›
El listado no se recarga automáticamente al crear registros. Presione Refrescar para volver a cargar los datos desde la base de datos.
¿Cómo puedo ver el detalle completo de un movimiento? ›
Haga doble clic sobre la fila. Se abrirá Frm_Encabezado_Transaccion con todos los datos precargados. Al cerrarlo, el listado se actualiza automáticamente.
¿Puedo editar los datos directamente en la tabla? ›
No. La cuadrícula tiene ReadOnly = true. Toda modificación debe realizarse abriendo el formulario de detalle con doble clic.
¿Qué hace el botón DashBoard / Filtrar? ›
Abre DSH_BRD_FRM, un panel de análisis con filtros avanzados para buscar movimientos por rango de fechas, tipo u otros criterios. Requiere el permiso Consultar.
El sistema muestra un error al abrir el formulario ›
Un MessageBox de error al iniciar generalmente indica un problema de conexión a la base de datos. Verifique que el servidor esté disponible y reporte el mensaje exacto al administrador de sistemas.
¿Cómo genero un reporte de los movimientos? ›
Haga clic en Imprimir (requiere permiso). Se abrirá Frm_ReporteMovimientoInventario donde podrá configurar parámetros, previsualizar y exportar o imprimir el informe.
