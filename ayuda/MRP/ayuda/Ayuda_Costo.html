Capacitación Costos Fases

Para el proceso de costo fases se selecciona la fase, el tipo y se ingresa el costo designado




Para eliminar, se selecciona un dato en el datagrid para posteriormente presionar el boton de eliminar, el cual pedirá confimación de borrado, la cual depende del usuario seleccionará si de verdad desea elimiar el registro.




Para actualizar se debe seleccionar un registro en la datagrid, una vez seleccionado se procede a modificar el campo deseado como puede ser el costo Una vez modificado se presiona el botón de editar y el campo será modificado con la nueva información
