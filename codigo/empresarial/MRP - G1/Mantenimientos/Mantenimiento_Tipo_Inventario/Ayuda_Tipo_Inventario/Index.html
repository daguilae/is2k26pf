Guía de Uso: Navegador de Producción

Este módulo permite administrar los Tipos de Inventario. A continuación, se detalla la función de cada botón de la barra de herramientas superior.

Funciones de los Botones

Acción Descripción
Ingresar / Nuevo Habilita los campos para registrar un nuevo tipo de inventario (ej. Materia Prima).
Modificar Permite editar el nombre del registro seleccionado actualmente.
Guardar Confirma y almacena los cambios realizados en la base de datos.
Cancelar Anula la operación actual y bloquea la edición de campos.
Eliminar Borra permanentemente el registro seleccionado de la tabla.
Consultar Abre la búsqueda para localizar un ID o Nombre específico.
Imprimir / Reporte Genera un documento PDF o vista de impresión de los tipos de inventario.
Navegación Usa los botones Inicio, Anterior, Siguiente y Fin para moverte entre los registros existentes.
Salir Cierra el formulario y regresa al menú principal.

Estructura de Datos (Tbl_Tipo_Inventario)

El sistema gestiona los siguientes campos de forma automática o manual:

  • Pk_Id_Tipo_Inventario: Código único e incremental del registro.
  • Nombre_Tipo_Inventario: Descripción del inventario. Los valores permitidos son:
    • Materia Prima
    • En Producción
    • Producto Final
Nota técnica: El motor de búsqueda utiliza InnoDB para asegurar la integridad referencial de los datos de inventario.
