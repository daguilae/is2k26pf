Ayuda: Registrar Estado de Fase de Producción

En este formulario se registran los estados que pueden tomar las fases de producción. Cada campo debe llenarse con información válida para que el registro se guarde correctamente.

1. ID del estado

Este campo acepta únicamente números enteros mayores a cero. No se permiten valores negativos, ceros ni decimales.

El ID no debe repetirse con otros IDs ya registrados en el sistema. Cada estado debe tener un identificador único.

Ejemplo: 1, 2, 3, 10

2. Nombre del estado

En este campo se ingresa el nombre que identifica al estado.

Registrar un nombre corto que identifique fácilmente el estado.

Ejemplo: Finalizada

3. Descripción del estado

Aquí se describe con mayor detalle el significado del estado.

La descripción sirve para explicar qué representa ese estado y cuándo se utiliza en el proceso.

Ejemplo: Estado para fases que ya finalizaron el proceso de producción.
Asegúrese de revisar cada campo antes de enviar el formulario. El ID debe ser único y mayor a cero, mientras que el nombre y la descripción deben ser textos claros y descriptivos.
