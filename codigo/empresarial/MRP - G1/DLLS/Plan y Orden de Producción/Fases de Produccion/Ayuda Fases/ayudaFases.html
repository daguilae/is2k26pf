Ayuda: Registrar Fases de Produccion

En este formulario se registran las fases de producción de un producto. Cada campo debe llenarse con información válida para que el registro se guarde correctamente.

1. Proceso para guardar

Para guardar una fase de producción debe seguir el proceso en orden:

  1. Seleccione un producto del listado de opciones.
  2. Escriba el nombre de la fase.
  3. Escriba la descripción de la fase.
  4. Ingrese la cantidad de horas hombre.
  5. Oprima el botón de guardar para almacenar el registro.

Debe llenar todos los campos para que el sistema permita guardar la fase de producción. El nombre y la descripción deben ser cadenas de texto. Las horas hombre deben ser un número entero mayor a 0.

Ejemplo: Fase "Montaje", Descripción "Fase de armado de componentes", Horas "8".

2. Proceso para actualizar

Para actualizar una fase de producción debe seguir estos pasos:

  1. Seleccione un producto del listado de opciones.
  2. Revise la tabla con las fases de producción registradas para ese producto.
  3. Seleccione la fila del registro que desea actualizar para que la información aparezca en los campos de texto.
  4. Modifique el nombre, la descripción o la cantidad de horas hombre.
  5. Presione el botón de modificar y confirme la actualización de los datos.
  6. La tabla se recargará para mostrar los datos actualizados.

El nombre y la descripción deben ser cadenas de texto. Las horas hombre deben ser un número entero mayor a 0.

3. Proceso para eliminar

Para eliminar una fase de producción debe seguir estos pasos:

  1. Seleccione un producto del listado de opciones.
  2. Revise la tabla con las fases de producción registradas para ese producto.
  3. Seleccione la fila del registro que desea eliminar para que la información aparezca en los campos de texto.
  4. Presione el botón de eliminar y confirme la eliminación del registro.
  5. La tabla se recargará para mostrar los datos actualizados.
Asegúrese de revisar el proceso antes de guardar, actualizar o eliminar. Nombre y descripción deben ser texto, y horas hombre deben ser un entero positivo.
